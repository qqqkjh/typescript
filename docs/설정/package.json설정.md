# package.json 설정값
##  package.json : Object 
* name : string
    * 필수값
    * 패키지 이름
* version : string
    * 필수값
    * 패키지 버전
* description : string
    * 패키지 설명
* scripts : Object
    * 패키지 전용 스크립트
    * ex) npm run test
* license : string
    * 패키지 배포 라이선스 명시
* dependencies : Object
    * 패키지 의존성   
* devDependencies : Object
    * 개발시에만 필요한 패키지 의존성
    * ex) npm install <패키지> --save-dev
